# Sentence Analyzer

This JavaScript project analyzes a sentence and provides the following counts:

- **Vowels**  
- **Consonants**  
- **Punctuation**  
- **Words**

It was built as part of my FreeCodeCamp exercises to practice **loops, conditionals, string manipulation, and basic JavaScript programming**.

---

## How to Use

1. Open the project in a code editor or browser console.  
2. Run the JavaScript file (e.g., `sentence-analyzer.js`).  
3. Enter a sentence when prompted.  
4. The program will display the number of vowels, consonants, punctuation marks, and words.
